# OSINT & Public Data Gathering

## 1. Search Engine Footprinting
*Technique: Using advanced operators to find sensitive files.*

### Google Hacking (Dorks)
* **Description:** Querying Google for vulnerable servers or exposed files.
* **GHDB:** [Google Hacking Database](https://www.exploit-db.com/google-hacking-database)
* **Useful Dorks:**
    * `site:target.com filetype:pdf` (Find public documents)
    * `site:target.com "password"` (Find leaked creds)
    * `intitle:"index of"` (Find open directories)

### IoT & Specialized Search
* **Shodan:** Search for connected devices (Webcams, Servers).
    * `city:"New York" org:"TargetCorp"`

## 2. Web Services & Social Sites
*Technique: Gathering info from LinkedIn, Job boards, and Repos.*

### Tools
| Tool | Purpose | Command / Link |

## DNS Reconnaissance

### dig
```bash
dig example.com ANY
dig example.com MX
dig @8.8.8.8 example.com
```

### nslookup
```bash
nslookup example.com
nslookup -type=mx example.com
```

### dnsenum
```bash
dnsenum example.com
```

### dnsrecon
> *HomePage: https://github.com/darkoperator/dnsrecon*
```bash
dnsrecon -d example.com -t std
```
